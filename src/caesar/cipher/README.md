# Криптоанализатор

## -----Описание проекта-----
Программа для шифрования/расшифровки текста, с использованием известного ключа.

### Основные режимы работы:
1. Шифровка текста с использованием известного ключа (Шифр Цезаря).
2. Расшифровка закодированного текста, с использованием известного ключа.

**Работа с файлами**
- Исходный текст для шифрования/расшифровки берем из файла (указывается путь к файлу)
- Так же указывается путь к файлу для записи зашифрованного/расшифрованного текста.
